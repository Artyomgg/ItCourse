Основные моменты при работе с JavaScript

Видео урок

JavaScript – язык, что применяется на стороне клиента. Все действия могут происходить без перезагрузки страницы пользователя. На нём можно реализовать всевозможные действия, например: создание часов на сайте, реализация анимации, создание эффектов, разработка полноценных веб приложений и т.д.

Современные веб технологии шагнули намного вперед. Сегодня можно используя JavaScript работать не только на стороне клиента, но также работать с сервером и базой данных. Благодаря платформе Node JS создание сайта стало возможным при использовании лишь одного языка программирования – JavaScript. Теперь можно не прописывать серверную часть сайта на PHP, Java или другом серверном языке.

В то же время стоит отметить, что больше 80% всех современных сайтов мира используют именно PHP в качестве серверного языка. Многие привыкли работать с PHP и не видят особой нужды в переходе на другие более современные технологии.

JavaScript по умолчанию включен во всех современных браузерах. Дополнительно вы можете проверить включен ли JS при помощи HTML тега noscript.
